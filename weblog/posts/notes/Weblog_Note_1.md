# Weblog: Note 1

## Feeds

I can put next tags in `<head></head>`:

- `{rss}` - will add an RSS feed.

- `{json}` - will add an JSON feed.

- `{atom}` - will add an Atom feed.

- `{feeds}` - will add all feeds listed above at once.

---

RSS:  

{rss}

Atom:  

{atom}

JSON:  

{json}

Feeds:  

{feeds}

---
