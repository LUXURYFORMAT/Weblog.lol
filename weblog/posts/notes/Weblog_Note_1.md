# Weblog: Note 1

## Feeds

I can put next tags in `<head></head>`:

- <code>{rss}</code> - will add an RSS feed.

- <code>{json}</code> - will add an JSON feed.

- <code>{atom}</code> - will add an Atom feed.

- <code>{feeds}</code> - will add all feeds listed above at once.

---

RSS:  

{rss}

Atom:  

{atom}

JSON:  

{json}

Feeds:  

{feeds}
